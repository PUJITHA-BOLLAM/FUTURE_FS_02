# FUTURE_FS_02
Mini CRM system

   A mini CRM system,which shows the functionality client lead management system or a contact form like how they add leads,delete leads and how a admin dashboard works on backend.
   
 ## Features
 - Modern UI
 - Lead Page
 - Admin Page
 - Admin Dashboard
 
  ## Technologies Used
  - HTML
  - CSS
  - JAVASCRIPT
